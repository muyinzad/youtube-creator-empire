# SECTION 1: Thumbnail Anatomy

## Perfect Thumbnail Structure

```
┌────────────────────────────────────────┐
│  ┌──────┐                              │
│  │ EMOTION │ ← FACE (exaggerated)     │
│  │  😊    │                            │
│  └──────┘                              │
│                                        │
│     CATEGORY LABEL                     │
│     ─────────────────                  │
│                                        │
│     MAIN HEADLINE TEXT                 │ ← KEY MESSAGE
│     (Bold, readable)                   │
│                                        │
│           [BRAND/WATERMARK]            │
└────────────────────────────────────────┘
```

## Key Elements

| Element | Purpose | Size |
|---------|---------|------|
| Face | Stop scroll | 30% of thumbnail |
| Text | Key message | 20% of thumbnail |
| Background | Context | 50% of thumbnail |
| Contrast | Readability | Use light/dark |
